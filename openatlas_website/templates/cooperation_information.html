Information regarding cooperations

In case you would like to establish a cooperation it will be very helpful to have some information on the planned project. Information doesn't need to be complete, exact nor is it binding. It just helps us to get a feeling for the project, requirements and dimensions. E.g if there will be thousands of files with a few gigabyte each.

Project

  • Name
  • Short identifier or acronym if available, e.g. MEDCON for Mapping Medieval Conflict
  • Principal Investigator(s)
  • Project start and duration
  • An abstract about the projects content and aims. Can also be a proposal draft, a website or similar.

Data

  • Will data be entered completely new or is there existing data to import?
  • What kind of information will be worked with: E.g. prosopographical data, geospatial data, texts, archaeological features, artifacts, files etc.
  • Estimate of data volume (how many entries) and file volume (Gigabyte)
  • How many people are expected to enter data

Organizational

  • Project's hosting institution
  • Type of grant or funding
  • Global project budget
  • Estimate of available budget for a cooperation
  • Will it utilize resources of an existing OpenAtlas project (e.g. infrastructure)

Which services are requested

  • Planning - e.g. evaluation how to map research data to CIDOC CRM and enter them into OpenAtlas
  • Hosting - e.g. installation and server maintenance
  • Support - e.g. software updates, training, troubleshooting
  • Development - e.g. software adaptions and additional features
  • Presentation website - if requested the more detailed requirements the better
  • Long time archiving in Arche

Feel free to contact us at openatlas@oeaw.ac.at.
